BOOTSTRAP

En nuestra pagina web deseamos utilizar como base principal el HTML, CSS y BOOTSTRAP, esto con el fin de poder aplicar todas las interfaces y utilidades de bootstrap. El tipo de animaciones que usariamos serian estas:

Animaciones de scroll: Estas animaciones responden al desplazamiento del usuario en la página web. Pueden utilizarse para crear efectos de paralaje, desvanecimiento, revelación de elementos y más.

Animaciones de carga: Se utilizan para mejorar la experiencia del usuario mientras se cargan elementos en la página web. Pueden incluir indicadores de carga, animaciones de carga progresiva y otros efectos visuales que mantienen al usuario comprometido mientras espera que se cargue el contenido.
